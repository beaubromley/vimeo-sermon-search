Quick Start (Recommended):
-------------------------
*** AUTOMATED PROCESS ***
Run the complete 4-step process automatically:
   python update_and_run.py

This will:
1. Download new videos and transcripts (incremental)
2. Update the searchable database
3. Show updated statistics
4. Launch the search interface automatically

The script automatically checks you're in the correct directory and 
handles errors gracefully. Just run it whenever you want to update!

Manual Process (Advanced Users):
-------------------------------
If you prefer to run steps individually:

1. Download Videos and Transcripts:
   python scripts/download_videos_incremental.py    (Fast - stops at first known video)
   OR
   python scripts/download_videos.py                (Thorough - checks all videos)

2. Process Transcripts into Database:
   python scripts/update_database.py
   - Creates searchable database from transcripts
   - Only processes new transcripts
   - Maintains existing database entries

3. View Statistics:
   python scripts/show_kpis.py
   - Shows comprehensive system statistics
   - Video counts by year, processing status
   - Storage usage and recommendations
   - Use --detailed flag for complete video list

4. Search Transcripts:
   python scripts/search_interface.py
   - Search through video titles AND transcript content
   - Shows timestamps and direct video links
   - Multiple search modes: titles only, transcripts only, or both
   - Press Ctrl+C or type 'quit' to exit

Troubleshooting:
---------------
If you get errors:
1. Make sure you're running from the vimeo_project root directory
2. Check that your Vimeo API credentials are correct
3. Verify internet connection
4. Check file permissions in the data directory
5. Make sure all required packages are installed
6. Run scripts/debug_transcript.py to test specific videos

If videos should have transcripts but don't download:
1. Check if transcripts are actually available on Vimeo
2. Run scripts/debug_transcript.py with the video ID
3. Try scripts/request_transcriptions.py to request auto-captions
4. Verify API permissions

File Locations:
